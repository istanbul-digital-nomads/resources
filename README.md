## Contributing

These guides are only as good as the community keeps them. If you notice outdated info or have something to add, please open a pull request or an issue. See our [contributing guide](https://github.com/istanbul-digital-nomads/community/blob/main/CONTRIBUTING.md).

## Disclaimer

Information in these guides reflects community members' personal experiences and may change over time. Always verify important details (especially visa rules and legal requirements) with official sources.
